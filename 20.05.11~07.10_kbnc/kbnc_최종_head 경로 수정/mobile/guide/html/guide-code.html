CSS 기본 규칙

  • 폰트의 선언, filter, content, background-image에 큰 따옴표("")를 사용한다.
  • 선택자, 중괄호, 콜론, 세미콜론 사이에 공백을 넣지 않는다.
  • 마지막 선언된 속성에도 세미콜론(;)을 사용한다.
  • 6자리 컬러코드가 동일할 시 3자리로 생략한다.
.class-name{font-family:"dotum";}
.filter:progid:DXImageTransform.Microsoft.AlphaImageLoader(src="img.png", sizingMethod="scale";}
.class-name:after{content:".";}
.class-name{color:#000;background:url("img.gif") no-repeat;}
