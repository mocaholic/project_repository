종류 구분 번호 예시

ID Selector

  • 최상위 레이아웃, 폼 요소, 책갈피를 제외하고 id를 사용하지 않는다.

예약어

예약어
구분 예약어 설명
사이트 레이아웃 #wrap 사이트 전체영역
#skip-nav 바로가기 영역
#header 머리글 영역
#container 페이지 영역
#aside 사이드 영역
#content 본문 영역
#footer 바닥글 영역
#pop-xxx 레이어팝업 페이지별 ID

Class Selector

  • 체인 선택자의 개수는 3개까지 권장한다.
  • 클래스 예약어는 상황에 맞게 선택적으로 참고한다.

예약어
구분 예약어 설명

.gnb 대표메뉴 (Global Navigation Bar)

이미지 네이밍 규칙

  • 종류별로 스프라이트 방식을 적용한다.
  • 이미지 네이밍의 조합은 '형태-의미-순서-상태'의 순서로 조합한다.
