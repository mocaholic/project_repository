KT RCS는 풍부하고 다양한 메시지를
전송 할 수 있습니다.

RCS SMS
액션버튼 1개 전송 가능한 단문 메시지
최대 90자 전송 가능.
RCS LMS
액션버튼 3개 전송 가능한 장문 메시지
최대 2000자 전송 가능.
RCS MMS
액션버튼 3개 전송 가능한 미디어 메시지
최대 2000자 전송 가능. 미디어 크기 1MB 까지 전송 가능.
Standalone
미디어를 위 또는 좌/우에 배치하여
풍부한 메시지로 구성해보세요.
Carousel
여러 개의 미디어를 한번에
발송하거나, 컨텐츠 내용의 구분이
필요할 때 사용할 수 있습니다.
템플릿
정보성 메시지 발송을 위한 템플리형 메시지 입니다.
나만의 템플릿을 생성하여 관리 할 수 있습니다.
Free 타입
"알림" 메시지 형태로 전송 합니다.
브랜드를 생성하면 자동 생성되는 템플릿 입니다.
별도 등록 없이 사용이 가능합니다.
Description 타입
용도에 따라 템플릿 양식을 선택하여
나만의 템플릿을 생성 후 사용이 가능합니다.
Cell 타입
용도에 따라 템플릿 양식을 선택하여
나만의 템플릿을 생성 후 사용이 가능합니다.
액션 버튼
버튼에 다양한 기능을 담아서 메시지를 전송 하세요.
URL 연결 웹이나 앱 으로 이동할 수 있습니다.
지도/ 위치 미리 지정된 지도의 위치를 보여주거나 사용자의 위치를 서버로 전송할 수 있습니다.
일정 등록 사용자의 캘린더에 특정 일정을 등록할 수 있습니다.
복사하기 특정 문구를 사용자 단말이 자동 복사 할 수 있게 합니다.
대화방 열기 다른 번호로 메시지를 보낼 수 있도록 대화방을 엽니다.
전화 연결 특정 전화번호로 전화를 걸 수 있습니다.
